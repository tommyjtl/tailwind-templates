# Tailwind Templates

A personal reference of common Tailwind CSS page layouts. Each template is pure HTML elements and Tailwind classes — no logic — so I can copy-paste a starting structure when kicking off a small work.

## Templates

| Template | Description |
|---|---|
| **Centralized Card** | Vertically and horizontally centered card with a form and list |
| **Dashboard** | Full-viewport shell with a fixed sidebar, top header, and scrollable content area |
| **Half & Half** | Split screen — centered card on the left, empty gray panel on the right |

## Stack

- React + TypeScript + Vite
- Tailwind CSS v4
